# 4. The furniture

**What this chapter settles:** how to tell which things in the restaurant are furniture, without confusing them with the things around them.

## One test keeps the category steady

Chapter 2 made sure everyone could name the same things. The restaurant now needs the same steadiness in what it contains. **Each thing belongs by a test, not by what somebody happens to call it.**

Furniture is anything you could carry out of the restaurant, but never use up. Both parts matter. Two further checks keep powered things and hand-held things in their own categories.

## Furniture can leave without becoming less

Tables, chairs and shelves are furniture. So are the pass counter and the benches chefs stand at. **You could carry each one out, and ordinary work does not use any of it up.**

A chair may wear out. A shelf may break. That does not mean either was used up. The test asks whether the ordinary work steadily leaves less of the thing behind. Furniture remains a whole thing until it is moved, replaced or broken.

## Four questions settle every doubtful thing

Ask these questions in order.

1. **Could you carry it out?** If not, it is not furniture.
2. **Does ordinary work use it up?** If so, it is not furniture.
3. **Is it powered to do work for you?** If so, it is equipment.
4. **Does a chef pick it up by hand to do the work?** If so, it is a utensil.

If the answer is yes to the first and no to the other three, put it in the furniture category. The test stays the same in every restaurant in the chain, because a category that changes with the person sorting it will not give you the same restaurant twice.

## The final checks keep four categories apart

The carrying test separates furniture from the building. The next two checks keep it apart from the other things in the restaurant.

**Furniture is not powered to do work for you.** If it is powered, it belongs with the equipment.

**Furniture is not picked up by hand to do the work.** If a chef picks it up to do the work, it belongs with the utensils. A bench can be carried out, but it stays in place while the work happens. That is why it is furniture.

This chapter only says that the furniture exists and where it belongs. How the restaurant uses it comes later.

---

## The rules, flat

*The same rules as above, with the explanation removed. This is the part the appliance
obeys.*

1. Class as furniture anything that could be carried out but is never used up in ordinary work.
2. Include tables, chairs, shelves, the pass counter and the benches chefs stand at.
3. Exclude anything that cannot be carried out.
4. Exclude anything that ordinary work uses up.
5. Exclude anything powered to do work; class it as equipment.
6. Exclude anything picked up by hand to do the work; class it as a utensil.
